    Section 4 Logical Arguments

    Objectives: Goals for This Section

    1. Define logical arguments.
      🔗
    2. Learn to verify whether a logical argument is valid by using truth tables.
      🔗
    🔗

    Subsection 4.1 Logical Arguments

    Definition 4.1.

    A logical argument consists of a list of several premises (statements that convey assumptions, rules, facts, and observations), followed by a unique statement called the conclusion of the argument.
    🔗
    🔗

    Example 4.2. Example of a Logical Argument.

    (Premise 1:) If you win the game, then I will celebrate.
    🔗
    (Premise 2:) You won the game.
    🔗
    (Conclusion:) Therefore, I am celebrating.
    🔗
    🔗

    Example 4.3. Another Example of a Logical Argument.

    (Premise 1:) If you are a hero, then you wear a Rolex.
    🔗
    (Premise 2:) You wear a Rolex.
    🔗
    (Conclusion:) Therefore, you are a hero.
    🔗
    🔗

    Definition 4.4.

    A valid argument is a logical argument in which, if all the premises are true, then the conclusion must also be true.
    🔗
    In other words, a valid argument means that true premises guarantee a true conclusion.
    🔗
    🔗

    Definition 4.5.

    A invalid argument is a logical argument in which all the premises can be true, but the conclusion can still be false.
    🔗
    An invalid argument is often called a fallacy.
    🔗
    🔗
    🔗

    Subsection 4.2 Truth Table Method

    We can use truth tables to verify whether a logical argument is valid. The method consists of the following steps:
    🔗
    1. Identify the premises and the conclusion.
      🔗
    2. Represent each premise and the conclusion in symbolic form.
      🔗
    3. Construct a table by placing the premises and the conclusion next to each other, with the last column representing the conclusion. Then extend the table to the left by adding columns for all necessary simple statements and intermediate expressions to complete the truth table.
      🔗
    4. Locate the rows where all premises are true, and then check the truth value of the conclusion in those rows.
      🔗
    If, in every row where all premises are true, the conclusion is also true, then the argument is valid.
    🔗

    Example 4.6. Example 1: A Valid Argument.

    If I go to a college, I will find a better job. I go to a college. Therefore, I will find a better job.
    🔗
    Let \(p\) = “I go to a college,” and \(q\) = “I will find a better job.”
    🔗
    Symbolic form:
    🔗
    \(p \to q\)
    🔗
    \(p\)
    🔗
    \(\therefore q\)
    🔗
    The only row in which both premises \(p \to q\) and \(p\) are true is the first row. In that row, the conclusion \(q\) is also true. Therefore, the argument is valid.
    🔗
    🔗

    Example 4.8. Example 2: A Valid Argument.

    If you love reading, you have a library membership. You do not have a library membership. Therefore, you do not love reading.
    🔗
    Let \(p\) = “You love reading,” and \(q\) = “You have a library membership.”
    🔗
    Symbolic form:
    🔗
    \(p \to q\)
    🔗
    \(\lnot q\)
    🔗
    \(\therefore \lnot p\)
    🔗
    The only row in which both premises \(p \to q\) and \(\lnot q\) are true is the last row. In that row, the conclusion \(\lnot p\) is also true. Therefore, the argument is valid.
    🔗
    🔗

    Example 4.10. Example 3: A Valid Argument.

    I will learn MATH 103 or I will watch Breaking Bad. I will not watch Breaking Bad. Therefore, I will learn MATH 103.
    🔗
    Let \(p\) = “I will learn MATH 103,” and \(q\) = “I will watch Breaking Bad.”
    🔗
    Symbolic form:
    🔗
    \(p \lor q\)
    🔗
    \(\lnot q\)
    🔗
    \(\therefore p\)
    🔗
    The only row in which both premises \(p \lor q\) and \(\lnot q\) are true is the second row. In that row, the conclusion \(p\) is also true. Therefore, the argument is valid.
    🔗
    🔗

    Example 4.12. Example 4: An Invalid Argument.

    If you are a hero, then you wear a Rolex. You wear a Rolex. Therefore, you are a hero.
    🔗
    Let \(p\) = “You are a hero,” and \(q\) = “You wear a Rolex.”
    🔗
    Symbolic form:
    🔗
    \(p \to q\)
    🔗
    \(q\)
    🔗
    \(\therefore p\)
    🔗
    In the third row, both premises \(p \to q\) and \(q\) are true, but the conclusion \(p\) is false. Therefore, this argument is invalid.
    🔗
    🔗

    Example 4.14. Example 5: Another Invalid Argument.

    If it is raining, then the ground is wet. The ground is wet. Therefore, it is raining.
    🔗
    Let \(p\) = “It is raining,” and \(q\) = “The ground is wet.”
    🔗
    Symbolic form:
    🔗
    \(p \to q\)
    🔗
    \(q\)
    🔗
    \(\therefore p\)
    🔗
    Notice that in the first row, both premises are true and the conclusion is also true. However, in the third row, both premises \(p \to q\) and \(q\) are true, but the conclusion \(p\) is false.
    🔗
    Since there is a row where the premises are true and the conclusion is false (T, T, F), the argument is invalid.
    🔗
    🔗
    🔗

    Subsection 4.3 Exercises

    Determine whether each of the following arguments is valid or invalid.
    🔗

    Question 4.16.

    If I were a chicken, then I would lay eggs. However, I am not a chicken, so I do not lay eggs.
    🔗
    Solution.
    Let \(p\) = “I am a chicken,” and \(q\) = “I lay eggs.”
    🔗
    Symbolic form:
    🔗
    \(p \to q\)
    🔗
    \(\lnot p\)
    🔗
    \(\therefore \lnot q\)
    🔗
    In the third row, both premises \(p \to q\) and \(\lnot p\) are true, but the conclusion \(\lnot q\) is false. Therefore, the argument is invalid.
    🔗
    🔗
    🔗

    Question 4.18.

    If interest rates go down, then I will buy a house. If I buy a house, I will need a loan. Therefore, I will not need a loan if I do not buy a house.
    🔗
    Solution.
    Let \(p\) = “Interest rates go down,” \(q\) = “I will buy a house,” and \(r\) = “I will need a loan.”
    🔗
    Symbolic form:
    🔗
    \(p \to q\)
    🔗
    \(q \to r\)
    🔗
    \(\therefore \lnot q \to \lnot r\)
    🔗
    In the seventh row, both premises \(p \to q\) and \(q \to r\) are true, but the conclusion \(\lnot q \to \lnot r\) is false. Therefore, the argument is invalid.
    🔗
    🔗
    🔗

    Question 4.20.

    If I am honest, you are lying; either you are lying or you have your facts wrong. You cannot have done your research, if you have your facts wrong. You did your research. Therefore, I am honest.
    🔗
    Solution.
    Let \(h\) = “I am honest,” \(l\) = “You are lying,” \(f\) = “You have your facts wrong,” and \(r\) = “You did your research.”
    🔗
    Symbolic form:
    🔗
    \(h \to l\)
    🔗
    \(l \lor f\)
    🔗
    \(f \to \lnot r\)
    🔗
    \(r\)
    🔗
    \(\therefore h\)
    🔗
    In the eleventh row, all the premises \(h \to l\text{,}\) \(l \lor f\text{,}\) \(f \to \lnot r\text{,}\) and \(r\) are true, but the conclusion \(h\) is false. Therefore, the argument is invalid.
    🔗
    🔗
    🔗

    Question 4.22.

    If you like apples, you will like this pie. If you like this pie, then you will like the bakery. Therefore, if you do not like the bakery, you do not like apples.
    🔗
    Solution.
    Let \(a\) = “You like apples,” \(p\) = “You like this pie,” and \(b\) = “You like the bakery.”
    🔗
    Symbolic form:
    🔗
    \(a \to p\)
    🔗
    \(p \to b\)
    🔗
    \(\therefore \lnot b \to \lnot a\)
    🔗
    In every row where both premises \(a \to p\) and \(p \to b\) are true, the conclusion \(\lnot b \to \lnot a\) is also true. Therefore, the argument is valid.
    🔗
    🔗
    🔗

    Question 4.24.

    Given the premises:
    🔗
    P1: If today is Saturday, then we do not have class today.
    🔗
    P2: If we do not have class today, then I will go shopping or I will go to the picnic.
    🔗
    (a) Write a valid conclusion (in English) for the argument.
    🔗
    (b) Write the entire argument in symbolic form.
    🔗
    (c) Use a truth table to justify whether the argument is valid or invalid.
    🔗
    Solution.
    Let \(s\) = “Today is Saturday,” \(c\) = “We have class today,” \(p\) = “I will go shopping,” and \(q\) = “I will go to the picnic.”
    🔗
    (a) A valid conclusion is:
    🔗
    If today is Saturday, then I will go shopping or I will go to the picnic.
    🔗
    (b) Symbolic form:
    🔗
    \(s \to \lnot c\)
    🔗
    \(\lnot c \to (p \lor q)\)
    🔗
    \(\therefore s \to (p \lor q)\)
    🔗
    (c) Construct a truth table with the premises next to each other and the last column as the conclusion.
    🔗
    Now look at the rows where both premises \(s \to \lnot c\) and \(\lnot c \to (p \lor q)\) are true. In each of those rows, the conclusion \(s \to (p \lor q)\) is also true.
    🔗
    Since there is no row in which all premises are true and the conclusion is false, the argument is valid.
    🔗
    🔗
    🔗
    🔗
